UPUTSTVO:
1. Kada budeš znao pravi domen, u index.html, robots.txt i sitemap.xml zameni:
   https://restoran-tradicija.rs/
   pravim domenom, npr. https://restorantradicija.rs/

2. Za Netlify:
   - Uploaduj ceo folder restoran_tradicija_production
   - Glavni fajl mora biti index.html

3. Za Google:
   - Dodaj domen u Google Search Console
   - Submituj sitemap.xml
   - Uradi URL Inspection za homepage
   - U Google Business Profile dodaj ovaj URL kao Website

4. U index.html postoje mesta za GTM, GA4 i Microsoft Clarity.
